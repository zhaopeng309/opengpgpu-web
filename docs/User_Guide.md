# OpenGPGPU 网站用户指导文档

欢迎使用 OpenGPGPU 官方网站系统。本文档旨在帮助最终用户（访客）快速了解并使用网站的前台功能，并帮助网站管理员掌握后台系统的日常维护和管理操作。

## 1. 简介

OpenGPGPU 官网是一个基于 Python Flask 框架开发的 Web 应用程序，采用 SQLAlchemy 进行数据库管理，包含前台信息展示页面以及后台管理系统。网站的主要功能包括发布项目最新公告、展示开发路线图（Roadmap）、提供技术文档入口以及连接开源社区。

## 2. 访客使用指南（前台系统）

普通用户和访客无需注册登录即可访问网站的所有前台页面：

### 2.1 首页 (Home)
- **访问路径**：`/` (网站根目录)
- **功能说明**：首页是 OpenGPGPU 的门户。您可以在这里快速了解项目的基本介绍和最新动态。首页会自动展示最新的项目公告列表，方便您追踪项目的重大更新。

### 2.2 路线图 (Roadmap)
- **访问路径**：`/roadmap`
- **功能说明**：展示 OpenGPGPU 项目的发展规划和进度。路线图数据通常按照时间（如季度）或里程碑进行分组，清晰地展示了项目过去取得的成就、正在进行的工作以及未来的计划。

### 2.3 官方文档 (Docs)
- **访问路径**：`/docs`
- **功能说明**：提供 OpenGPGPU 的技术文档、开发指南和 API 参考等资源的统一入口，帮助开发者和研究人员深入了解项目的技术细节。

### 2.4 社区 (Community)
- **访问路径**：`/community`
- **功能说明**：展示 OpenGPGPU 开源社区的相关信息，包含论坛链接、代码仓库地址（如 GitHub/Gitee）、邮件列表以及其他交流渠道，欢迎您参与社区互动和贡献。

## 3. 管理员使用指南（后台系统）

后台系统仅供拥有管理员权限的人员使用，用于维护网站的内容（如公告、路线图等）。

### 3.1 管理员登录
- **访问路径**：`/admin/login`
- **操作步骤**：
  1. 在浏览器地址栏输入完整的后台登录 URL。
  2. 在登录界面输入您的管理员用户名和密码。
  3. 点击“登录”按钮。如果凭证正确，系统将自动跳转至后台仪表盘（Dashboard）。

### 3.2 控制台 / 仪表盘 (Dashboard)
- **访问路径**：`/admin/dashboard`
- **功能说明**：登录成功后的默认主页。在这里，您可以总览网站当前的核心数据。
  - **公告管理**：在仪表盘或专属的公告管理页面，您可以查看所有的公告记录，执行添加新公告、编辑现有公告或删除过期公告等操作。
  - **系统登出**：当完成管理工作后，请点击页面上的“登出”或“Logout”按钮（对应路径 `/admin/logout`），以确保您的账户安全。

*(注：系统架构上已设计了基于 `AdminService`、`AnnouncementService` 和 `RoadmapService` 的数据处理能力，实际可管理的模块以仪表盘显示的侧边栏或功能按钮为准。)*

## 4. 开发者与运维指南（快速启动）

如果您是网站的维护人员或需要进行本地部署测试，请参考以下简要流程：

### 4.1 环境准备
1. 确保系统已安装 Python 3.x 环境。
2. 推荐使用虚拟环境：
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/macOS
   # 或 venv\Scripts\activate # Windows
   ```

### 4.2 安装依赖
在项目根目录下，运行以下命令安装项目所需的 Python 依赖包：
```bash
pip install -r requirements.txt
```

### 4.3 初始化数据库
系统使用 SQLite 作为默认数据库。在第一次运行项目前，需要初始化数据库表结构：
```bash
python init_db.py
```
*此操作会在 `instance/` 目录下生成 `opengpgpu.db` 数据库文件。*

### 4.4 运行服务
您可以直接使用 Flask 内置服务器启动项目（仅供开发测试使用）：
```bash
python wsgi.py
```
默认情况下，服务会在 `http://127.0.0.1:5000` 启动。

### 4.5 环境变量配置
您可以根据项目根目录的 `.env` 文件（或创建该文件），配置自定义的密钥（`SECRET_KEY`）、数据库连接字符串（`SQLALCHEMY_DATABASE_URI`）等环境变量。
